## About

This repository documents my journey through the **Complete Intro to React Native** on Frontend Masters. The course builds a strong foundation in React Native by exploring components, props, state, hooks, mobile navigation patterns, and native rendering concepts using TypeScript.

### Project Context

**Purpose**: Build production-ready cross-platform mobile applications from first principles  
**Scope**: Full course curriculum including exercises and the capstone mobile app  
**Instructor**: [Kadi Kraman](https://frontendmasters.com/teachers/kadi-kraman/)

This repository serves as a hands-on laboratory for understanding React Native’s rendering model, component composition, state-driven mobile UI architecture, and type-safe application development.

---

## Key Learnings

Throughout this course, I implemented:

- **Native Component Architecture**: Building reusable UI using `View`, `Text`, `ScrollView`, and custom components.
- **Props & Data Flow**: Managing one-way data flow between components.
- **State Management**: Handling dynamic UI updates with `useState` and TypeScript types.
- **Side Effects**: Fetching and synchronizing API data with `useEffect`.
- **Navigation**: Implementing stack and tab navigation with React Navigation.
- **Mobile Styling**: Designing responsive layouts using `StyleSheet` and Flexbox.
- **React Native Internals**: Understanding native rendering, the JavaScript bridge, and performance considerations.
